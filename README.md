# dmd-mouse-human-rnaseq

This repository contains analysis code supporting the study:

**Type I Interferon Mediates Trained Immunity and Muscle Weakness in Duchenne Muscular Dystrophy Mice**

The codebase includes reproducible pipelines and notebooks for **cross-species (mouse and human) RNA-seq analysis in Duchenne muscular dystrophy (DMD)**, covering bulk RNA-seq, single-cell RNA-seq, and single-nucleus RNA-seq datasets.

<!--
## Study Overview

Duchenne muscular dystrophy (DMD) is characterized by progressive muscle degeneration accompanied by chronic inflammation and oxidative stress. In dystrophic (mdx) muscle, macrophages exhibit persistent innate immune hypersensitivity, a phenomenon consistent with trained immunity. However, the molecular drivers of trained immunity and their contribution to muscle dysfunction in DMD remain poorly understood.

In this study, we show that macrophage ontogeny is severely disrupted in mdx mice, with resident macrophages undergoing apoptosis and being replaced by bone marrow–derived macrophages exhibiting a trained immune phenotype. This phenotype is associated with exaggerated type I interferon signaling, resembling classical antiviral responses. These findings are supported by cross-species analyses, including human DMD RNA-seq datasets.

Genetic inhibition of type I interferon signaling reverses the trained macrophage phenotype, reduces oxidative stress, and improves diaphragm muscle force-generating capacity. Systemic delivery of a decoy receptor for type I interferons similarly improves exercise performance. Together, these results identify type I interferon–mediated trained innate immunity as a promising therapeutic target in DMD.
